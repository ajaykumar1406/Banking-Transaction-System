# Banking Transaction System

## Description
A Java-based console application that simulates basic banking operations such as
account creation, deposit, withdrawal, balance enquiry, and transaction history.

## Technologies Used
- Java
- JDBC
- MySQL
- Git & GitHub

## Features
- User authentication
- Deposit and withdrawal
- Balance check
- Transaction history
- Menu-driven console interface

## How to Run
1. Import the project into any Java IDE
2. Configure MySQL database
3. Run the main class
